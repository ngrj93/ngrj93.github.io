Projects & Publications

Foresight (Ongoing) - This application aims at helping users to schedule their visits by suggesting days that are less likely to be crowded. Users are able to select a time period within which they would like to visit an establishment/POI. The application then presents projections of expected footfall on days within the range and suggests suitable days based on certain user preferences. UWP

Pastry DHT (2016) - Implemented a full-fledged Pastry distributed object location service for finding and sharing data over a distributed peer network. C++

Pintos (2016) - This project involved the design of a CPU scheduling mechanism in a barebones instructional operating system framework. C

Morpheus (2015-2016) - A research project following a grass roots approach to the development of wireless ad hoc high performance computing clusters with the aim of achieving an architecture for ubiquitous computing. C++/C

GoldHunter (2015) - A data mining tool implementing various pattern recognition algorithms (Apriori, FP-growth, DB-Scan, K-means clustering) for analysing data. C#

e-Gurukul (2014) - An intranet based academic discussion forum and digital resource library management system for students and teachers. PHP
